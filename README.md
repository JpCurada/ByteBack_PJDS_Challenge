# ByteBack_Mariano_Curada_Notebook

## Overview

This repository contains a Jupyter Notebook titled "ByteBack_Mariano_Curada_Notebook" that demonstrates a Python data science project. The notebook includes data analysis, visualization, and machine learning tasks.

## Prerequisites

Before running the notebook, ensure you have the following prerequisites installed on your system:

- Python 3.x: You can download Python from [python.org](https://www.python.org/downloads/).
- Jupyter Notebook: Install Jupyter Notebook using pip with the command: `pip install jupyter`.
